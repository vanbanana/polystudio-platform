# 营销素材模板

本目录存放可复用的模板文件：

## 文件类型

- **海报模板**: 海报设计源文件参考（PSD/Canva等）
- **图文模板**: 微信公众号/小红书图文版式
- **视频模板**: 短视频脚本模板/分镜模板
- **邮件模板**: EDM/营销邮件HTML模板

## 使用示例

可直接复制以下模板结构到实际内容中。

### 微信文章模板

```
【标题】
副标题

【正文分段1】
内容...

【正文分段2】
内容...

【结尾引导】
作者/品牌介绍
```

### 短视频脚本模板

```
[时长]
[画面内容]
[台词/解说]
[音效/BGM]
[动作]

---
```

### 广告落地页模板

```
[首屏]
标题 + 副标题 + CTA

[信任] 
资质/数据/评价

[转化]
促销信息 + 表单/按钮
```
