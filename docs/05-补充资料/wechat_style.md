# 微信公众号写作指南

## 公众号类型与风格

### 企业号（服务号/订阅号）

- 风格：专业、真诚、有价值
- 重点：服务性质内容 + 品牌调性展示
- 更新频率：建议每周1-3篇

### 个人IP号

- 风格：个性化、有观点、有温度
- 重点：个人IP打造 + 垂直领域内容
- 更新频率：建议每周3-5篇

## 标题规范

### 最佳标题长度
- 主标题：20-28字
- 副标题：8-12字（如有）

### 标题类型

| 类型 | 示例 | 适用场景 |
|------|------|----------|
| 数字型 | 5个方法帮你xxx | 干货类 |
| 疑问型 | xxx是如何做到的？ | 揭秘类 |
| 对话型 | 告诉你一个秘密 | 亲近感 |
| 冲突型 | 原来xxx一直是错的 | 颠覆认知 |
| 热点型 | xxx官方回应了 | 热点借势 |

### 标题雷区
- 避免过度标题党（如"震惊！"、"99%的人都不知道"）
- 避免夸张失实
- 避免生硬堆砌关键词

## 正文结构

### 开头Hook（至关重要）

前150字决定用户是否继续阅读：

```
✓ 场景代入：我相信你一定遇到过...
✓ 痛点共鸣：是不是每次xxx都...
✓ 数据吸引：90%的人都不知道...
✓ 悬念设置：xxx的真相是...
```

### 内容框架

1. **引入**（100-200字）
   - 场景描绘 or 痛点提出

2. **正文**（800-1500字）
   - 层次分明，小标题区分
   - 每段不超过3行手机显示宽度

3. **总结**（100-200字）
   - 要点回顾
   - 行动号召

4. **互动**（50-100字）
   - 引导评论、点赞、在看

## 排版规范

### 字间距
- 推荐：1.5-2px

### 行间距
- 推荐：1.5-1.8倍

### 段落间距
- 推荐：空一行

### 重点突出
- 加粗关键词
- 适当使用背景色强调
- 列表使用有序/无序符号

### 配图规范
- 封面图：16:9 (900x500px 最佳)
- 文中配图：宽度600-900px
- 建议添加品牌水印

## 违禁词与敏感词

### 极限词（广告法禁止）

"最"、"第一"、"顶级"、"国家级"、"世界级"等

### 敏感行业词（需资质）

医疗、药品、保健品、金融、投资等

### 投诉敏感词

"免费"、"赠品"、"返利"等（可能触发恶意投诉）

## 数据参考

- 最佳发布时间：工作日 12:00-13:00、18:00-21:00
- 周末 10:00-12:00、20:00-22:00
- 标题字数：20-28字为佳
- 摘要字数：30-50字
- 阅读完成率目标：30%以上
