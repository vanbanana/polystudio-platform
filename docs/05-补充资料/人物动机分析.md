# 人物动机分析框架

## 动机层次模型

### 表层动机（External Goal）
角色公开追求的目标，外在可观察
- 例子：找到失踪的父亲、证明自己的清白、揭开真相

### 深层动机（Internal Need）
角色内心真实的渴望或恐惧，往往不自知
- 例子：渴望被认可、害怕被抛弃、追求正义感

### 隐藏秘密（Secret）
角色拼命掩盖的事实，是驱动行为的关键
- 例子：失手伤人、隐瞒身份、知道真相

## 动机冲突类型

1. **目标冲突**：角色A的目标与角色B的目标直接对立
2. **价值冲突**：角色对"正确"的理解根本不同
3. **身份冲突**：角色必须扮演与其本性矛盾的角色
4. **情感冲突**：理性判断与情感需求产生矛盾

## 悬疑剧中的人物设计要点

### 不可信叙述者
- 角色有理由隐瞒或扭曲信息
- 观众需要"发现"真相

### 双重身份
- 表面身份与真实身份不同
- 身份揭露是重要的反转点

### 灰色角色
- 不存在绝对的好人与坏人
- 每个角色都有其行为逻辑

### 动机递进
- 早期：表层动机驱动
- 中期：深层动机显现
- 结局：动机的终极考验
