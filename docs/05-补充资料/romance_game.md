# 恋爱养成模板

## 适用场景

适合创作以角色互动、情感发展为核心的互动内容，玩家通过选择影响与角色的亲密度或攻略对象。

## 核心机制

| 机制 | 说明 |
|------|------|
| 好感度 | 选择影响数值，影响剧情走向 |
| 攻略对象 | 可单恋/多选/多攻略 |
| 事件触发 | 特定条件触发隐藏事件 |
| FLAG | 选择埋设伏笔，影响结局 |

---

## 剧情分支

### 【START】：「相遇」
【场景】
在xxx地点，你第一次见到xxx...

【选项】
A. 主动搭话 → 【角色A线-初见】
B. 默默观察 → 【角色B线-初见】

---

### 【角色A线-初见】：「命运的邂逅」
【场景】
xxx...

【好感变化】
- 选A：角色A +10
- 选B：角色A +5

【选项】
A. [进一步互动] → 【角色A-事件1】
B. [保持距离] → 【公共剧情】

---

### 【角色A-事件1】：「约会事件」
【场景】
xxx（增加互动机会）

【隐藏FLAG】
- 如果好感度≥60：触发【隐藏事件】

【选项】
A. ... → 【分支】
B. ...

---

## 结局类型

| 结局 | 条件 | 描述 |
|------|------|------|
| HE | 好感度≥80达成结局 | 甜蜜在一起 |
| NE | 好感度50-79 | 友谊以上恋人未满 |
| BE | 好感度<30或选项触发 | 关系破裂/错过 |
| 隐藏TE | 隐藏FLAG+特殊选项 | 真结局 |
