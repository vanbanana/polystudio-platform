# 心理测试模板

## 适用场景

适合创作趣味测试、性格分析、选择题互动等内容，玩家通过选择获得对应结果，增加互动性和分享欲。

## 核心机制

| 机制 | 说明 |
|------|------|
| 计分系统 | 选项对应不同分值，累积计分 |
| 维度分析 | 多维度测试更丰富 |
| 权重计算 | 不同选项对不同结果有不同权重 |
| 复选结果 | 允许并列/多种结果 |

```markdown
## 计分规则

### 维度定义
| 维度 | 说明 | 分值范围 |
|------|------|----------|
| 维度A | [如：外向] | 0-10 |
| 维度B | [如：内向] | 0-10 |
| 维度C | ... | ... |

### 选项分值
- A选项：维度A +2，维度B +0
- B选项：维度A +1，维度B +1
- C选项：维度A +0，维度B +2
```

## 结果判定

### 分数计算方式
总分 = 各维度分值相加
主导维度 = 最高分维度

### 结果对照

| 主导维度 | 结果类型 | 结果标题 | 描述 |
|----------|----------|----------|------|
| 维度A | 类型1 | 「XX型」 | 特征描述... |
| 维度B | 类型2 | 「XX型」 | 特征描述... |
| 维度C | ... | ... | ... |

### 混合结果（进阶）
- 维度A + 维度B ≥ X：复合类型「XX+XX型」
- 存在极端值：隐藏类型

## 爆款测试特征

1. **话题性强**：戳中痛点/痒点
2. **结果有趣**：超出预期/自我验证
3. **分享动力**：彰显个性/引发讨论
4. **社交货币**：测试结果可作为谈资
