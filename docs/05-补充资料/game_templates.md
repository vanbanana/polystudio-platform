# 互动游戏模板汇总

## 模板选择指南

| 游戏类型 | 推荐模板 | 分支特点 |
|----------|----------|----------|
| 文字冒险 | text_adventure.md | 线性+少量分支，重剧情 |
| 恋爱养成 | romance_game.md | 网状分支，重角色互动 |
| 悬疑推理 | mystery_game.md | 树状分支，重逻辑解谜 |
| 心理测试 | quiz_game.md | 线性计分，重结果多样性 |
| 剧情模拟 | sim_game.md | 多线并行，重沉浸感 |

---

## 通用元素模板

### 标题格式

```
# 《故事标题》——副标题（可选）
## 类型标签：【XX】｜【XX】
```

### 角色卡片模板

```markdown
## 角色设定

### 主角
- **姓名**：[名字]
- **年龄**：[年龄]
- **性格**：[性格关键词x3]
- **背景**：一句话背景

### 关键配角（可选）
- **角色名**：xxx
- **定位**：xxx（导师/对手/伙伴/暗恋对象）
- **关键特征**：xxx
```

### 节点模板

```markdown
### 【节点ID】：「节点标题」

【场景描述】
[环境描写+人物状态+当前危机/机遇]

【可选行动】
A. [行动描述] → 触发【节点ID】
B. [行动描述] → 触发【节点ID】
C. [行动描述] → 触发【结局ID】

【分支提示】
[可选：暗示选择后果的伏笔]
```

### 结局模板

```markdown
### 【结局ID】：「结局标题」

【结局类型】
- HE（Happy Ending）：幸福结局
- NE（Normal Ending）：普通结局
- BE（Bad Ending）：bad结局
- TE（True Ending）：真结局（需特殊条件解锁）

【结局内容】
[结局描述]

【隐藏要素】（可选）
[如：隐藏结局触发条件]
```

---

## 分支复杂度参考

| 篇幅 | 节点数 | 结局数 | 适用场景 |
|------|--------|--------|----------|
| 短篇 | 5-8 | 2-3 | 单次阅读/测试 |
| 中篇 | 10-20 | 4-6 | 连载/深度互动 |
| 长篇 | 25+ | 7+ | 完整作品 |

---

## 快速选择建议

- **新手创作**：从「短篇+双结局」开始
- **测试类内容**：选择 quiz_game.md 模板
- **故事向**：选择 text_adventure.md 或 romance_game.md
- **悬疑向**：选择 mystery_game.md，加入解谜要素
