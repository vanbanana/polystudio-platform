# 营销素材生成器

## 概述

本技能帮助生成各类营销素材，覆盖从社交媒体到广告投放的全渠道内容需求。

## 核心工作流程

### 1. 需求分析

生成营销素材前，先明确以下要素：

- **产品/服务**：推广的是什么
- **目标受众**：谁会看到这条内容
- **投放渠道**：内容将在哪个平台展示
- **营销目标**：品牌认知、转化、互动还是留存
- **风格调性**：专业严肃、轻松活泼、高端精致还是亲民实惠
- **篇幅要求**：短文案（50字内）、中文案（100-300字）还是长文案（500字以上）

### 2. 内容生成

根据需求选择对应的内容类型进行生成：

```
内容类型选择：
├── 社交媒体文案
│   ├── 微信公众号文章/推文
│   ├── 微博帖子/长图文案
│   ├── 抖音/快手短视频脚本
│   ├── 小红书种草笔记
│   └── B站视频简介/专栏
├── 广告文案
│   ├── 信息流广告（今日头条、腾讯广告等）
│   ├── 搜索广告（百度、搜狗等）
│   └── 户外广告/电梯广告
├── 产品文案
│   ├── 产品卖点提炼
│   ├── 产品详情页文案
│   └── 竞品对比文案
├── 营销邮件
│   ├── 促销活动邮件
│   ├── Newsletter/周报
│   └── 客户关怀邮件
└── 品牌内容
    ├── 品牌故事
    └── 海报图文/九宫格
```

### 3. 优化建议

生成初稿后，可进行以下优化：

- **标题党检测**：避免过度夸张，保持可信度
- **关键词密度**：确保核心关键词出现1-3次
- **行动号召 (CTA)**：明确引导用户下一步动作
- **emoji使用**：根据平台特性适量添加
- **字数控制**：根据平台限制调整长度

## 各平台内容规范

### 微信生态

- **公众号标题**：32字以内，避免标题党
- **正文**：建议分段，每段不超过3行
- **封面图**：16:9 比例，建议添加 logo 水印
- **引导关注**：文末添加关注引导语

### 抖音/快手

- **视频标题**：建议25字以内，带话题标签
- **文案**：建议55字以内（前5字吸睛）
- **热门标签**：按内容添加3-5个相关话题

### 小红书

- **标题**：建议18字以内，可加 emoji
- **正文**：建议300-500字，段落清晰
- **标签**：添加3-5个精准标签

### 微博

- **普通微博**：建议120字以内，重点前置
- **长图微博**：可发多张图片配文案
- **热搜借势**：快速蹭热点，抢占先机

### B站

- **视频简介**：建议100字以内
- **专栏文章**：支持长篇，可添加目录
- **弹幕/评论互动**：鼓励用户弹幕互动

## 常用营销框架

### AIDA 模型

```
A (Attention) 注意力 - 用痛点/悬念/数据吸引注意
I (Interest) 兴趣    - 延续好奇心，引入产品
D (Desire) 欲望     - 展示卖点，打动用户
A (Action) 行动     - 明确引导，促成转化
```

### 痛点-方案-效果 模型

```
1. 描述目标用户的痛点场景
2. 介绍产品/服务如何解决
3. 展示使用后的美好效果
```

### 3B 原则（广告创意）

```
Beauty 美丽 - 展示美好的视觉
Beast 动物 - 借用可爱的动物
Baby 婴儿 - 温馨的家庭感
```

### 悬念+解答 结构

```
[抛出悬念/问题] → [逐步展开] → [揭示答案/解决方案]
```

## 常用英文缩写

| 缩写 | 全称 | 含义 |
|------|------|------|
| CTA | Call To Action | 行动号召 |
| USP | Unique Selling Proposition | 独特卖点 |
| KOL | Key Opinion Leader | 关键意见领袖 |
| KOC | Key Opinion Consumer | 关键意见消费者 |
| UGC | User Generated Content | 用户生成内容 |
| ROI | Return On Investment | 投资回报率 |
| CTR | Click Through Rate | 点击率 |
| CVR | Conversion Rate | 转化率 |
