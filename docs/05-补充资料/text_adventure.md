# 文字冒险模板

## 适用场景

适合创作具有完整剧情线的互动故事，玩家通过选择推动剧情，体验不同走向的阅读体验。

## 模板结构

```markdown
# 《标题》——副标题
## 【文字冒险】｜【XX题材】

---

## 世界观

### 背景设定
[时代/地点/世界观核心设定 - 100字内]

### 核心冲突
[故事的主要矛盾/危机]

---

## 角色

### 主角
- **姓名**：[名字]
- **身份**：[社会角色]
- **目标**：[核心诉求]

### 关键角色（1-3人）
- **角色名**：定位 - 性格特点
```

---

## 写作技巧

### 开场钩子（黄金300字）
- 制造悬念/危机/疑问
- 快速建立代入感
- 避免冗长背景介绍

### 选项设计原则
- 每个选项代表不同价值观/策略
- 选项间要有明显区分度
- 选项文案要有画面感

### 节奏把控
- 3-5节点设置一个小高潮
- 核心反转放在中段
- 结局前给玩家"以为要HE了"的假象
