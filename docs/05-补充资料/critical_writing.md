# 批判性写作指南

## 什么是批判性写作

批判性写作不是"批评"或"否定"，而是对已有研究进行**理性分析**：
- 评估研究质量
- 识别优点与不足
- 识别研究空白
- 思考改进方向

---

## 批判性分析的维度

### 1. 理论层面

| 分析维度 | 提问 |
|----------|------|
| 理论深度 | 理论基础是否扎实？概念界定是否清晰？ |
| 理论适用性 | 理论假设是否合理？是否有局限性？ |
| 理论创新 | 是否有新的理论贡献？ |
| 理论对话 | 是否与现有理论对话？ |

### 2. 方法层面

| 分析维度 | 提问 |
|----------|------|
| 研究设计 | 研究设计是否合理？能否回答研究问题？ |
| 样本选择 | 样本是否有代表性？样本量是否充足？ |
| 测量工具 | 测量是否有效？信效度如何？ |
| 数据分析 | 分析方法是否恰当？是否报告效应量？ |

### 3. 实证层面

| 分析维度 | 提问 |
|----------|------|
| 研究发现 | 结论是否由数据充分支持？ |
| 结果推广 | 结论是否可以推广到其他情境？ |
| 实际意义 | 研究发现是否有实践价值？ |
| 效应大小 | 效应是否具有实际意义？ |

---

## 批判性表达技巧

### 强批判（明确指出问题）

- "该研究在方法上存在明显缺陷..."
- "研究结论缺乏足够的实证支持..."
- "这一解释未能考虑...因素..."

### 中性批判（指出改进空间）

- "研究方法具有一定局限性..."
- "研究结论的推广性有待进一步验证..."
- "虽然在...方面进行了探索，但...方面仍有待深入..."

### 柔和批判（以建议呈现）

- "未来研究可以考虑..."
- "如果能够进一步...，研究将更加完善..."
- "后续研究可扩大样本范围..."

## 批判性阅读清单

### 阅读时记录

- [ ] 这篇研究的核心论点是什么？
- [ ] 支撑论点的证据有哪些？
- [ ] 研究方法是否恰当？
- [ ] 研究结论是否有局限性？
- [ ] 与其他研究相比有何异同？
- [ ] 对我的研究有何启发？

### 写作时自查

- [ ] 是否只是简单罗列文献，而没有分析？
- [ ] 是否对每篇文献都进行了评价？
- [ ] 是否识别了研究之间的关联与矛盾？
- [ ] 是否指出了现有研究的主要贡献？
- [ ] 是否清晰界定了研究空白？

---

## 常见批判性短语

### 肯定性评价

- 做出了重要贡献
- 为...提供了有力证据
- 在...方面具有创新性
- 推进了对...的理解

### 建设性批评

- 存在一定局限
- 仍有改进空间
- 需要进一步验证
- 有待深入探讨

### 指出空白

- 缺乏...的研究
- 尚未探明...
- 仍有待解答...
- 需要进一步考察...

## 批判性写作示例

### ❌ 非批判性（流水账式）

```
王明（2015）研究了A对B的影响，发现A对B有显著影响。
李华（2016）也研究了A对B的影响，得出了类似结论。
张伟（2017）研究了C对D的影响，发现C对D有影响。
```

### ✅ 批判性（分析式）

```
王明（2015）开创性地探讨了A对B的影响，并发现显著正相关。
然而，该研究仅基于某高校学生样本（n=200），其结论的外部效度
有待验证。李华（2016）虽采用更大样本（n=800），但仍旧局限
于单一地区。相比之下，张伟（2017）虽然在C对D的影响方面有所
进展，但其研究未控制潜在的混淆变量。综合来看，现有研究在...
方面仍存在明显空白（具体说明），这为本文的研究提供了切入点。
```

## 练习建议

1. **文献研讨**：与同伴讨论文献的好处与不足
2. **批注训练**：阅读时习惯性做批判性批注
3. **写作反思**：写完后检查是否只是"综"而没有"述"
4. **对比学习**：比较优秀综述与自己写作的差距
