## 一、打脸章节模板

### 典型结构

| 部分 | 字数 | 内容 |
|------|------|------|
| 铺垫 | 300-500 | 矛盾出现，主角被轻视/被打压 |
| 发酵 | 300-500 | 围观者起哄，对方变本加厉 |
| 真相 | 300-500 | 主角暴露实力/身份 |
| 反打 | 500-800 | 完虐对方，吃瓜群众震惊 |
| 收尾 | 200-300 | 收获/影响/新悬念 |

### 案例

```
【铺垫】
"陈北玄，你这个废物也敢来参加宗门大比？" 李昊挡住去路。
"滚开。"陈北玄淡淡道。
"哟，还装逼？"李昊大笑，"你们看，这个废物以为他还是三年前的天才呢！"
周围一片哄笑。
三年前，陈北玄是家族天才；现在经脉寸断，连炼气一层都没有。

【发酵】
"当年的天才，现在连给我提鞋都不配。"李昊优越感十足，"识相的跪下磕头，我考虑让你滚得体面点。"
"聒噪。"陈北玄皱眉。
"敬酒不吃吃罚酒！"李昊挥手就是一拳，"今天教你做人！"

【真相】
拳风凛冽，直取面门。
陈北玄不闪不避，伸出两根手指，轻松夹住。
"怎么可能……"李昊变色。
"你太慢了。"陈北玄摇头，"三年了，你还是这点本事。"
体内灵气运转，筑基期威压爆发。

【反打】
"筑基期！？"
"他不是废了吗？"
"不可能！"
围观者炸锅了。
陈北玄轻描淡写地将李昊甩出去，后者砸碎三张桌子，半天爬不起来。
"现在，谁还有意见？"
无人应答。

【收尾】
长老席位上，大长老霍然站起："此子……必须重点培养！"
（完）
```

## 二、升级/突破章节模板

### 典型结构

| 部分 | 字数 | 内容 |
|------|------|------|
| 困境 | 200-300 | 当前修炼遇到的瓶颈/危机 |
| 转折 | 300-500 | 获得机遇/灵感/宝物 |
| 冲关 | 500-800 | 艰难的过程描写 |
| 突破 | 300-500 | 升级成功，灵异景象 |
| 收获 | 200-300 | 新能力的展示/战力对比 |

### 案例

```
【困境】
"还是不行……"
感受着体内堵塞的经脉，xxx苦笑。
三个月了，卡在炼气九层不得寸进。
宗门大比在即，以现在的实力，连前十都进不了。

【转折】
"可曾想过，换条路？"
脑海中，神秘声音响起。
"你是谁？"xxx警惕。
"我是谁不重要。重要的是，我知道一门功法，可重塑经脉。"
"什么代价？"
"万年灵髓一滴。"
xxx咬牙："，成交。"

【冲关】
灵气如决堤洪水，疯狂涌入。
疼，疼入骨髓！
经脉寸寸断裂，又重新生长。
xxx死死咬住舌尖，保持最后一丝清明。
不能晕，晕了就完了！

【突破】
"轰！"
体内一声轰鸣。
突破了！
筑基期！
天空骤变，乌云汇聚，竟是雷劫！
"，区区雷劫，也敢阻我？"
xxx长身而起，直面天威！
第一道劫雷，碎！
第二道劫雷，碎！
第三道……

【收获】
三道劫雷过，xxx全身焦黑，但气息反而更强。
感受着体内澎湃的力量，xxx露出笑容。
"筑基中期……这功法果然逆天！"
"Now，来看看战力提升了多少。"
（完）
```

## 三、暧昧/感情线章节模板

### 典型结构

| 部分 | 字数 | 内容 |
|------|------|------|
| 接触 | 300-500 | 与女方产生交集/互动 |
| 升温 | 300-500 | 暧昧氛围，肢体接触 |
| 危机 | 300-500 | 情敌出现/误会/危险 |
| 保护 | 300-500 | 主角英雄救美 |
| 推进 | 200-300 | 关系微妙变化 |

### 案例

```
【接触】
"小心！"
纤腰被揽住，xxx带着怀中人翻滚出去。
箭矢擦肩而过。
"你……放开我。"美女满脸通红。
"噢。"xxx这才发现姿势少儿不宜，触手温软。
赶紧松开。

【升温】
"谢谢你救了我。"楚楚可怜的表情，"我无以为报……"
"不用，举手之劳。"
"不行！"美女坚持，"至少让我请你吃顿饭。"
盛情难却。
饭桌上，得知她叫林诗雨，是林家千金。

【危机】
"诗雨！"
一个公子哥走来，脸色阴沉。
"他是谁？"
"我朋友。"林诗雨躲到xxx身后。
"朋友？呵。"公子哥冷笑，"小子，你最好离她远点。"
"如果我说不呢？"

【保护】
"敬酒不吃吃罚酒！"
公子哥挥手，身后保镖围上来。
xxx叹气："看来这顿饭没那么容易吃完。"
三分钟后。
公子哥和他的保镖躺了一地。
"还有谁想让我离她远点？"
无人应答。
林诗雨看向xxx的眼神不一样了。

【推进】
"你……到底是什么人？"
"江湖散人。"
"骗子。"林诗雨轻笑，"哪有这么强的散人。"
"你不也骗我说是普通千金？"
两人相视一笑。
（完）
```

## 四、战斗/副本章节模板

### 典型结构

| 部分 | 字数 | 内容 |
|------|------|------|
| 开场 | 200-300 | 敌人出现，局势介绍 |
| 试探 | 300-500 | 初次交锋，互相试探 |
| 激烈 | 500-800 | 底牌尽出，战况激烈 |
| 转折 | 300-500 | 反转/主角反击 |
| 结束 | 200-300 | 胜负已分，掉落/收获 |

### 案例

```
【开场】
"滚出万魔渊，饶你们不死。"
为首的黑袍人拦住去路，元婴期威压扑面而来。
xxx脸色凝重。
己方最强的长老也只是元婴初期。
一场恶战，在所难免。

【试探】
"结阵！"
长老当机立断，十人组成剑阵。
"雕虫小技。"黑袍人冷笑，随手拍出。
惊天气浪碰撞。
剑阵摇摇欲坠，对方却只退了三步。
实力差距，太大了。

【激烈】
"我拖住他，你们走！"长老咬牙。
"走？"黑袍人杀意凛冽，"一个都别想走！"
杀招频出。
长老全身是血，但一步不退。
"前辈！"xxx眼眶欲裂。
"臭小子……还愣着干什么！走啊！"
"不。"
xxx站到长老身边。
"，今日一起死战！"
"你疯了！你只是金丹！"

【转折】
"金丹，谁说金丹不能杀元婴？"
xxx体内沉寂的血脉，突然沸腾。
"系统，燃烧所有灵气！"
"【警告：将付出巨大代价】"
"我同意！"
气势暴涨。
金丹后期……金丹巅峰……元婴初期！
"这气息……不可能！"
黑袍人变色。

【结束】
"这就是……元婴的力量？"
黑袍人跪地，满是不可置信。
"不可能……绝对不可能……"
"没什么不可能。"
xxx收剑。
"回去告诉你们主上，这笔账，我会亲自去讨。"
（完）
```

## 五、高潮/大场面章节模板

### 典型结构

| 部分 | 字数 | 内容 |
|------|------|------|
| 铺垫 | 300-500 | 大战前的气氛/准备 |
| 渲染 | 500-800 | 双方阵容描写 |
| 爆发 | 500-800 | 战斗进入白热化 |
| 最高潮 | 300-500 | 主角逆袭/绝招 |
| 结局 | 200-300 | 胜负与影响 |

### 案例

```
【铺垫】
三日后，万里高空。
密密麻麻的修士悬浮各方，天机阁、逍遥宗、万魔宫……
十大宗门齐至。
只是为了绞杀一人。
"陈北玄，交出帝经，饶你魂魄转世！"
天机阁主声音如雷。

【渲染】
"就凭你们？"
陈北玄持剑而立，身后只有七道身影。
八大势力，上百名化神期。
兵力悬殊。
但他不怕。
"今日，要么我死，要么你们全部陪葬！"
剑鸣声响彻云霄。

【爆发】
"杀！"
喊杀震天。
灵光爆炸，法宝横飞。
陈北玄如入无人之境，剑下无一合之敌。
鲜血染红长空。
但敌人太多了。
身上伤口不断增加。
"还不死？"天机阁主皱眉。
"呵……"陈北玄擦去嘴角血迹，"热身结束。"

【最高潮】
"系统，融合所有技能。"
"【警告：此操作不可逆，是否确认？】"
"确认。"
气息疯狂攀升。
化神初期……化神中期……化神巅峰……渡劫期！
"这气息……不可能！"
"这世上，没有什么不可能。"
一剑斩出。
天地色变。
天机阁主，形神俱灭！

【结局】
"还有谁？"
陈北玄环顾四方。
无人敢应。
"滚。"
一个字，各大宗主退去。
（完）
```

## 六、日常/水字数章节模板

### 典型结构

| 部分 | 字数 | 内容 |
|------|------|------|
| 日常片段1 | 300-400 | 生活小事/互动 |
| 日常片段2 | 300-400 | 另一个场景 |
| 小插曲 | 300-400 | 小冲突/小事件 |
| 解决 | 200-300 | 圆满解决/感情升温 |
| 过渡 | 200-300 | 引出后续剧情 |

### 注意事项

- 日常章不能太水
- 每段要有信息量
- 最后要引出后续剧情
- 不要连续写超过3章日常

## 七、断章技巧汇总

### 断章位置

**推荐断章：**
- 高潮高潮处（"就在这千钧一发之际——"）
- 反转处（"他笑了，原来一切都在计划中。"）
- 悬念处（"门外，传来敲门声。"）
- 收获处（"属性点+10，可分配——"）

**避免断章：**
- 平铺直叙处
- 无关紧要的对白
- 节奏拖沓的描写

### 断章话术

```
"想学吗？我教你啊。"
……
（断章）

"这功法……"
老者脸色大变，"你是从哪里得到的！？"
……
（断章）

门开了。
出现在门后的，赫然是——
……
（断章）

"其实，我等这一天，已经等了三千年。"
……
（断章）

而此刻，正在发生变化的，不止他一个。
……
（断章）
```
