## 引用格式规范

### GB/T 7714-2015 格式示例

**期刊论文**：
```
[序号] 作者. 文献题名[J]. 刊名, 出版年份, 卷号(期号): 起止页码.
[1] 王明, 李华. 人工智能在教育中的应用[J]. 教育研究, 2020, 41(3): 45-53.
```

**专著**：
```
[序号] 作者. 书名[M]. 出版地: 出版者, 出版年份.
[2] 张三. 高等教育管理研究[M]. 北京: 教育科学出版社, 2019.
```

**会议论文**：
```
[序号] 作者. 文献题名[C]//会议论文集. 出版地: 出版者, 出版年份: 起止页码.
```

### APA格式（第7版）示例

**期刊论文**：
```
作者, A. A., & 作者, B. B. (年份). 文章标题. 期刊名称, 卷号(期号), 页码. https://doi.org/xxxxx
```

**APA 7 变化**：
- 作者姓名前不需要缩写点
- DOI 取代 URL 作为主要引用来源
- APA不再区分纸质/网络期刊

## 注意事项

### 文献综述查重

| 期刊类型 | 常见查重率要求 |
|----------|----------------|
| 中文核心 | ≤15%（单篇） |
| 中文普刊 | ≤20% |
| SCI/SSCI | 视期刊而定 |

**建议**：
- 将多篇文献的综合观点用自己的语言表达
- 避免连续引用同一文献超过100字
- 正确标注所有引用来源

### 伦理要求

1. **系统性综述**：建议在 PROSPERO 注册
2. **Meta分析**：需遵循 PRISMA 报告规范
3. **知情同意**：如涉及人类受试者研究需声明
4. **伦理审批**：动物/人类研究需注明审批单位
