# 网文类型套路大全

本文档整理主流网文类型的经典套路，供创作时参考。

---

## 一、玄幻/仙侠类

### 1.1 经典套路汇总

**废物逆袭流**
- 主角本是废柴（灵根缺失/经脉堵塞/被陷害）
- 意外获得金手指（老爷爷/系统/传承/宝物）
- 逐步崛起，报仇雪恨
- 爽点：打脸曾经的轻视者

**退婚流**
- 未婚妻/家族上门退婚
- 主角觉醒隐藏血脉/实力
- 当众打脸
- 后续：未婚妻后悔/反追

**系统流**
- 主神/系统/游戏化设定
- 发布任务 → 获得奖励
- 变强途径清晰，便于情节推进

**学院/宗门流**
- 进入修炼宗门/学院
- 升级考试/比试
- 结识天骄/红颜
- 争夺资源

**家族流**
- 家族天才/少主
- 家族内斗/继承人之争
- 带领家族崛起

**洪荒流**
- 穿越到盘古开天后
- 夺取先天灵宝/跟脚
- 量劫中挣扎求存

### 1.2 境界设定参考

**修真体系（传统）：**
```
炼气 → 筑基 → 金丹 → 元婴 → 化神 → 渡劫 → 仙人
```

**玄幻体系（西方奇幻）：**
```
青铜 → 白银 → 黄金 → 铂金 → 钻石 → 王者 → 至尊
```

**简化体系：**
```
初级 → 中级 → 高级 → 顶级 → 尊者 → 圣人 → 神
```

### 1.3 升级奖励维度

每次升级可以给予主角：
- 新技能/功法
- 身体素质提升
- 特殊能力/特效
- 身份地位变化
- 获得宝物
- 势力归属变化

## 二、都市类

### 2.1 经典套路汇总

**神豪流**
- 系统/继承/抽奖获得大量金钱
- 疯狂消费/投资
- 打脸拜金女/富二代
- 建立商业帝国

**兵王流**
- 退役特种兵/杀手/佣兵
- 保护总裁/千金
- 各路美女倒贴
- 解决各种麻烦

**修仙归来流**
- 都市修仙者
- 隐姓埋名/扮猪吃虎
- 遇到故人/仇家
- 重修/恢复实力

**直播流**
- 直播日常/探险/带货
- 网红之路
- 各界关注

**钓鱼/捡漏流**
- 系统提示哪里有宝
- 赌石/鉴宝/捡漏
- 专家打脸

**医术流**
- 继承神医传承
- 疑难杂症手到病除
- 权贵求医

### 2.2 都市爽点来源

- 金钱数字带来的爽感
- 打脸拜金/势利眼
- 美女主动靠近
- 仇家跪舔
- 地位提升
- 能力超常

## 三、历史类

### 3.1 经典套路汇总

**魂穿历史人物**
- 穿越成汉武帝/唐太宗/朱元璋等
- 改变历史走向

**发明创造流**
- 搞出玻璃/水泥/火药
- 推动社会发展

**科举仕途流**
- 十年寒窗
- 金榜题名
- 官场沉浮

**争霸天下流**
- 招兵买马
- 逐鹿中原
- 一统天下

### 3.2 历史文注意事项

- 考据要严谨（经典段落可以查资料）
- 语言风格要符合时代
- 避免过度YY
- 了解朝代特色

---

## 四、悬疑/推理类

### 4.1 经典套路汇总

**密室杀人**
- 经典不可能犯罪
- 现场留下的线索

**心理悬疑**
- 犯罪心理学
- 侧写师/行为分析

**系统/游戏悬疑**
- 逃生游戏
- 死亡直播
- 限时破案

**灵异悬疑**
- 鬼打墙/灵异事件
- 捉鬼/除妖

### 4.2 悬疑写作技巧

- 线索要公平（读者能猜到）
- 误导要合理
- 结尾要反转
- 气氛渲染要到位

---

## 五、轻小说/二次元

### 5.1 经典套路汇总

**恋爱日常**
- 从相识到相恋的过程
- 各种撒糖/修罗场

**校园恋爱**
- 高中/大学校园
- 社团活动
- 青春成长

**综漫/综武**
- 穿越多个动漫/武侠世界
- 收集/兑换

**转生类**
- 转生异世界
- 开后宫/种田/冒险

### 5.2 轻小说特点

- 语言轻松幽默
- 内心吐槽多
- 人物互动有趣
- 情感细腻
- 节奏明快

---

## 六、套路运用原则

### 6.1 套路不是复制

套路是经过验证的爽点模式，但要用出自己的特色：
- 结合自己的设定做调整
- 加入个人风格的元素
- 注意不要踩毒点

### 6.2 常见毒点（避免踩雷）

- 圣母心过度
- 主角被绿
- 无意义虐主
- 战力崩坏
- 太监烂尾
- 主角窝囊

### 6.3 创新方向

在经典套路上创新：
- 类型混搭（都市+修仙）
- 设定反转让（退婚的变成了主角）
- 视角创新（反派/配角视角）
- 时代背景创新（现代+古代融合）
