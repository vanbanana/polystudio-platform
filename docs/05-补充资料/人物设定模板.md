# 人物设定模板库

本文档提供各类角色设定模板和案例。

---

## 一、主角设定模板

### 模板1：经典废柴逆袭流

```markdown
## 主角：叶凡

### 基础信息
- 性别：男
- 年龄：18
- 身份：叶家庶出子弟 -> 散修 -> （待定）
- 性格：沉稳内敛 / 人不犯我我不犯人 / 护短

### 外在
- 常用穿着：粗布麻衣，背后有一把剑
- 标志性动作：拇指抹过剑柄
- 标志性特征：左脸有一道疤（儿时被暗算留下）
- 口头禅："剑下不斩无名之鬼。"

### 内在
【过去】
- 童年：父亲战死，母亲被家族排挤
- 阴影：经脉天生堵塞，无法修炼
- 转折：10岁被赶出家族，流落街头

【现在】
- 最在意：母亲/恩人/曾经帮助过他的人
- 弱点：涉及母亲的底线会被触动
- 习惯：每天寅时起床练剑（雷打不动）

【追求】
- 短期目标：进入天剑宗修炼
- 终极目标：找到父亲死亡的真相

### 金手指
- 类型：剑仙传承+系统辅助
- 获得：意外掉落山崖，遇到陨落的剑仙残魂
- 限制：每天只能签到一次领奖励
- 成长：剑意→剑气→剑罡→剑域

### 人物弧光
- 起点：自卑的废物
- 转折：获得传承，第一次杀人，第一次杀人
- 终点：剑道巅峰，守护想守护的一切
```

### 模板2：都市神豪流

```markdown
## 主角：陈北玄

### 基础信息
- 性别：男
- 年龄：25
- 身份：社畜 -> 神豪
- 性格：表面上唯唯诺诺 / 内心腹黑 / 极为护短

### 外在
- 常用穿着：普通衣服（钱多了也不张扬）
- 标志性动作：推眼镜（稍微有点慌的时候）
- 标志性特征：笑容温和，但让人发毛
- 口头禅："这样不好吧？" / "我这个人很老实的。"

### 内在
【过去】
- 童年：孤儿院长大
- 阴影：被领养后又被抛弃，认为自己不被需要
- 转折：28岁仍然一事无成，女友嫌贫爱富分手

【现在】
- 最在意：孤儿院的孩子/院长
- 弱点：有关孤儿院的任何负面消息
- 习惯：每个月匿名给孤儿院打钱

【追求】
- 短期目标：先赚一个小目标
- 终极目标：建立世界上最大的慈善帝国

### 金手指
- 类型：神豪系统（财富兑换系统）
- 获得：被女友分手那天，激活系统
- 限制：每小时只能用一次，每次最多转出一万
- 成长：随着地位提升，额度增加（可投资/可捐赠）

### 人物弧光
- 起点：自卑的普通人
- 转折：获得财富后迷失过一段时间
- 终点：找到比钱更重要的东西（人的尊严/感情）
```

### 模板3：穿越者/系统流

```markdown
## 主角：林轩（穿越者）

### 基础信息
- 性别：男
- 前世：地球程序员 / 网文老书虫
- 今世：灵天宗外门弟子
- 性格：咸鱼 / 喜欢吐槽 / 关键时刻靠谱

### 前世记忆
- 看过大量网文，熟悉各种套路
- 有现代思维，不迂腐
- 常有"我知道剧情走向"的上帝视角

### 金手指
- 类型：穿越自带的地球知识 + 系统
- 系统功能：
  - 任务系统：发布各种作死任务
  - 抽奖系统：完成任务获得积分
  - 商城系统：买各种奇怪道具

### 处事风格
- 能苟就苟，绝不浪
- 没事不惹事，有事不怕事
- 对敌人心狠手辣，对自己人护短
```

## 二、配角设定模板

### 模板4：红颜/伴侣型

```markdown
## 女主角：苏清雪

### 基础信息
- 性别：女
- 年龄：20
- 身份：冰云宫少宫主
- 性格：外冷内热 / 冰雪聪明 / 认定就不放手

### 外在
- 穿着：白色纱裙，冰蓝色配饰
- 气质：拒人于千里之外的冷艳
- 标志性：永远带着面纱（因为长得太美）

### 与主角关系
- 初见：主角救了她（假小子造型，没发现是女的）
- 发展：在秘境中患难与共
- 感情线：日久生情 -> 双向暗恋 -> 表白

### 人物特点
- 实力：同辈中顶尖
- 特长：阵法 / 冰系功法
- 弱点：不会表达感情 / 傲娇
- 记忆点：只有在主角面前才会露出温柔的一面
```

### 模板5：兄弟/基友型

```markdown
## 配角：王胖子

### 基础信息
- 性别：男
- 年龄：22
- 身份：散修 -> 主角兄弟
- 性格：话唠 / 贪财 / 关键时刻不掉链子

### 外在
- 特点：胖乎乎 / 贼眉鼠眼
- 口头禅："老叶啊"、"有钱途" / "要发"

### 与主角关系
- 认识：市集上因为一枚灵石结缘
- 友情：过命的兄弟
- 作用：负责搞笑 / 提供情报 / 衬托主角

### 人物高光
- 虽然贪婪但有底线
- 关键时候能豁出命
- 每次主角出事都是他通风报信
```

### 模板6：师父/贵人型

```markdown
## 配角：张老（戒指老爷爷）

### 基础信息
- 性别：男（灵魂体）
- 身份：数千年前的顶级强者
- 性格：腹黑 / 毒舌 / 嘴硬心软

### 被困原因
- 当年冲击更高境界失败
- 灵魂躲进戒指
- 等待有缘人

### 与主角互动
- 毒舌：主角修炼慢会各种嫌弃
- 护短：外人欺负主角会暴走
- 傲娇：帮了忙不承认

### 金手指功能
- 修炼指导（顶级眼光）
- 功法典籍（见多识广）
- 人脉关系（认识很多牛人）
```

## 三、反派设定模板

### 模板7：前期小反派

```markdown
## 反派：李昊

### 基础信息
- 性别：男
- 身份：叶家嫡系子弟
- 性格：欺软怕硬 / 纨绔 / 没脑子

### 与主角冲突
- 原因：曾经羞辱过主角
- 升级：主角崛起后，不断找麻烦
- 结局：被打脸后怀恨在心，投靠更强势力

### 作用
- 初期提供爽点（打脸）
- 引出后续势力
- 展示主角成长

### 特点
- 有后台所以嚣张
- 每次出来都被打脸
- 像个跳梁小丑
```

### 模板8：中期BOSS

```markdown
## 反派：血魔宗少主

### 基础信息
- 性别：男
- 身份：血魔宗唯一继承人
- 性格：阴险 / 残忍 / 工于心计

### 与主角冲突
- 原因：血魔宗灭主角全族
- 升级：多次阴谋对付主角
- 对比：智商比前期反派高很多

### 实力定位
- 略强于主角
- 有底蕴（宗门传承）
- 有底牌（轻易不会暴露）

### 功能
- 中期主要对手
- 推动主角升级
- 丰富世界观（展示敌对势力）
```

### 模板9：最终BOSS

```markdown
## 终极反派：天魔帝

### 基础信息
- 性别：男（/无性）
- 身份：域外天魔入侵者
- 实力：此界最强

### 塑造要点
- 不是单纯坏，有自己的理念
- 与主角有思想碰撞
- 最终一战要有仪式感

### 特点
- 活了无数年的老怪物
- 对主角有某种关注
- 可能有过类似经历

### 主角vs反派的核心冲突
- 不是简单的杀父之仇
- 是理念/道路的冲突
- 会让读者思考谁对谁错
```

## 四、设定技巧

### 让人物立住的技巧

1. **记忆点**：每个人物至少有一个能让读者记住的特征
2. **差异化**：同一类人设要有区分度
3. **一致性**：性格决定行为，不要OOC
4. **成长性**：配角也可以有弧光

### 人物关系矩阵

建议创建主要人物关系图：

```
        主角
       / | \
   师父 兄弟  红颜
   |     |      \
配角A  配角B    情敌
   |     |        \
反派1  反派2    更大反派
```

### 著名配角模板参考

| 类型 | 模板 | 记忆点 |
|------|------|--------|
| 傲娇女主 | 冰山美人会害羞 | 反差萌 |
| 忠犬男二 | 默默守护不求回报 | 深情 |
| 毒舌师父 | 嘴硬心软 | 傲娇 |
| 逗逼兄弟 | 负责搞笑 | 调节气氛 |
| 阴险反派 | 智商在线 | 有压迫感 |
| 正派长辈 | 公正但古板 | 立场不同 |

## 五、快速生成工具

### 生成器公式

```
主角模板 = 过去的创伤 + 现在的追求 + 金手指 + 性格特征
反派模板 = 与主角的冲突点 + 智商水平 + 行为动机 + 最后结局
配角模板 = 功能定位 + 与主角关系 + 记忆点 + 性格特征
```

### 组合建议

**最常见的组合：**
- 主角：废柴 + 获得金手指 + 沉稳护短
- 红颜：高冷女神 + 傲娇 + 日久生情
- 兄弟：话唠逗比 + 贪财 + 关键时刻靠谱
- 师父：隐藏强者 + 毒舌 + 护短
- 反派：不断找麻烦 + 每次都输 + 背后有人

**创新的可能：**
- 主角不是废柴而是天才（但有其他困境）
- 红颜不是倒追而是宿敌
- 兄弟不是逗比而是高冷
- 师父不是帮助而是利用（暗黑系）
