# 学术文献综述写作助手

## 概述

本技能帮助研究人员撰写高质量的学术文献综述，专注于中文核心期刊、SCI、SSCI等学术论文中的文献综述部分。涵盖文献检索、筛选、分析、写作全流程。

## 核心工作流程

### 1. 课题分析与主题确定

在开始文献综述前，明确以下要素：

- **研究主题**：你的研究问题是什么
- **研究领域**：所属学科及细分方向
- **目标期刊**：准备投稿的期刊（中文核心/CSSCI/SCI/SSCI）
- **时间范围**：近5年？近10年？还是更有历史跨度
- **文献类型**：仅期刊论文？是否包含专著、会议论文、学位论文

### 2. 文献检索策略

```
文献检索渠道：
├── 中文数据库
│   ├── CNKI（中国知网）
│   ├── 万方数据
│   └── 维普网
├── 外文数据库
│   ├── Web of Science
│   ├── Scopus
│   ├── PubMed
│   └── Google Scholar
└── 其他资源
    ├── 导师/同行推荐
    ├── 参考文献追溯
    └── 预印本平台
```

**检索词构建**：
- 核心关键词（中英文）
- 同义词、近义词
- 上位词、下位词
- 布尔逻辑组合（AND/OR/NOT）

### 3. 文献筛选与管理

**筛选标准**：
- 相关性：与研究问题直接相关
- 权威性：核心期刊、高被引、知名作者
- 时效性：近期研究为主，经典文献为辅
- 质量：研究方法规范、数据可靠

**文献管理工具**：
- EndNote
- Zotero
- Mendeley
- NoteExpress

### 4. 文献综述结构框架

#### 4.1 经典结构：年代主题式

```
└── 引言
    └── 研究背景与意义
└── 主体部分
    ├── 早期研究（年份区间）
    │   └── 主要发现与理论
    ├── 中期发展（年份区间）
    │   └── 研究进展与争论
    └── 近期趋势（年份区间）
        └── 最新研究方向
└── 总结与评述
    └── 研究空白与本研究定位
```

#### 4.2 主题式结构

```
└── 引言
└── 主体部分
    ├── 主题一：理论框架
    │   ├── 观点A
    │   └── 观点B
    ├── 主题二：实证研究
    │   ├── 研究方法
    │   └── 研究结论
    └── 主题三：争议与分歧
        └── 各方观点对比
└── 研究空白
```

#### 4.3 方法论式结构

```
└── 主体部分
    ├── 定量研究文献综述
    ├── 定性研究文献综述
    └── 混合研究文献综述
```

#### 4.4 国内/国外对比式

```
└── 主体部分
    ├── 国内研究现状
    │   ├── 理论视角
    │   └── 实证发现
    └── 国外研究现状
        ├── 理论视角
        └── 实证发现
```

### 5. 写作规范

#### 5.1 语言要求

| 要素 | 要求 |
|------|------|
| 措辞 | 学术严谨，避免口语化 |
| 时态 | 过去时描述已完成研究，现在时描述客观事实 |
| 人称 | 客观第三人称为主 |
| 语气 | 谨慎保守，避免绝对化表述 |

#### 5.2 常见表达句式

**引出文献**：
- "XX（年份）研究表明……"
- "已有学者（XX, 年份）指出……"
- "在XX领域，众多学者（XX, 年份；XX, 年份）进行了深入研究……"

**综合评述**：
- "综合来看，现有研究在……方面取得了重要进展"
- "然而，在……问题上仍存在争议/研究不足"

**指出研究空白**：
- "尽管如此，关于……的研究仍较为匮乏"
- "现有研究主要关注……，而对……的关注相对不足"

#### 5.3 避免的问题

- ❌ 简单罗列文献（流水账式）
- ❌ 缺乏批判性分析
- ❌ 文献堆砌，缺乏逻辑主线
- ❌ 遗漏重要文献
- ❌ 引用来源不准确

### 6. 常见问题诊断

| 问题症状 | 可能原因 | 解决方案 |
|----------|----------|----------|
| 文献堆砌，没有主线 | 缺乏理论框架 | 引入理论视角组织文献 |
| 前后重复，内容冗余 | 分类不清 | 按主题/维度重新组织 |
| 只综不述，缺乏深度 | 分析深度不足 | 增加批判性评价 |
| 文献陈旧 | 检索策略有误 | 补充近2-3年文献 |
| 与研究问题脱节 | 主题聚焦不够 | 明确研究问题再筛选 |

## 各学科文献综述特点

### 社会科学类

- 强调理论对话
- 需呈现研究脉络
- 注重批判性分析
- 适当使用图表可视化

### 理工科类

- 方法导向更强
- 技术演进时间线
- 实验方法对比
- 参数/结果数据化呈现

### 医学/生物学类

- 遵循PRISMA指南
- 证据级别分层
- 纳入/排除标准
- 系统评价/Meta分析

## 常用写作框架

### 漏斗式结构

```
广泛背景 → 缩窄至研究领域 → 具体研究问题 → 本研究定位
```

### 矩阵式分析

创建文献对比矩阵：

| 研究 | 理论视角 | 方法 | 样本 | 主要发现 | 局限性 |
|------|----------|------|------|----------|--------|
| 作者A | XX理论 | 问卷 | N=500 | 结论1 | 局限1 |
| 作者B | YY理论 | 实验 | N=100 | 结论2 | 局限2 |

### 研究空白识别法

```
Step 1: 识别已有研究"已做了什么"
Step 2: 识别"还未做什么"
Step 3: 明确"为什么需要做"
Step 4: 定位本研究贡献
```

## 注意事项

1. **文献时效性**：中文核心期刊通常要求近5年文献占比>50%
2. **引用规范**：严格遵循目标期刊的参考文献格式（GB/T 7714、APA、MLA等）
3. **重复率**：文献综述部分需保持原创性，避免过度引用单篇文献
4. **伦理声明**：系统性综述需注册并声明纳入排除标准
5. **语言润色**：投稿前建议进行专业英语润色（如针对SCI/SSCI）
