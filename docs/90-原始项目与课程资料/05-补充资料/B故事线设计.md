# B 故事线设计

## 什么是 B 故事线

B 故事线是依附于主线（A 故事线）的副线剧情，通常围绕：
- 情感关系（爱情、友情、亲情）
- 个人成长
- 配角的故事

---

## A/B 故事线的关系

### 表层关系
- B 故事线独立于 A 故事线发展
- 两个故事发生在不同场景/人物之间

### 深层关系
- B 故事线反映 A 故事线的主题
- 两条线索在关键节点交织
- B 故事线揭示 A 故事线角色的内心

---

## B 故事线模板

### 爱情线模板
```
起点：两个角色的初始关系（陌生/敌对/暗恋）
障碍：身份对立/误会/外力阻挠
转折：共同经历危机，关系拉近
危机：情感与理性的冲突（爱情 vs 责任）
高潮：角色必须做出选择
解决：B 故事线与 A 故事线互相成就
```

### 友情线模板
```
起点：友谊的基础与默契
挑战：利益冲突或信任危机
背叛：友情受到严重考验
真相：误解解开，友谊升华
牺牲：朋友为对方做出牺牲
```

### 亲情线模板
```
起点：亲情的纽带（或缺失）
裂痕：过去的创伤/误解
寻找：一方寻求修复关系
真相：揭示隐藏的真相/苦衷
和解：关系的重建或告别
```

---

## B 故事线的功能

### 情感调节
- 缓和悬疑的紧张气氛
- 让人物更加立体

### 信息传递
- 通过 B 故事对话透露信息
- 侧写主要人物

### 主题深化
- B 故事反映核心主题
- 提供"如果"的思考

### 结局支撑
- B 故事的解决为结局增添情感厚度

---

## 在悬疑剧中设计 B 故事线

### 常见模式

1. **爱情作为诱饵**
   - 利用爱情关系制造误会
   - 情感纠葛增加角色复杂性

2. **友情成为线索**
   - 朋友的身份是谜题关键
   - 友谊面临真相的考验

3. **亲情是动机来源**
   - 为主角行动提供情感动力
   - 家人卷入阴谋

### B 故事线的悬念设置

- 情感走向的不确定性
- 关系是否能够维系
- 谁会为爱做出牺牲

---

## 集数与 B 故事线节奏

| 阶段 | 集数 | B 故事线任务 |
|------|------|-------------|
| 引入 | 1-3集 | 呈现关系基础，埋下伏笔 |
| 发展 | 4-8集 | 障碍出现，关系拉锯 |
| 深化 | 9-15集 | 真相揭露，关系转折 |
| 高潮 | 16-20集 | 做出选择，情感危机 |
| 结局 | 21-30集 | 关系解决，与 A 线呼应 |
