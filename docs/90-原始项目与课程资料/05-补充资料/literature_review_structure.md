# 文献综述结构模板

## 一、引言部分（通常占全文10-15%）

### 内容要素

1. **研究背景**
   - 研究领域的重要性
   - 现实/理论问题的产生背景
   - 简要交代问题的来龙去脉

2. **研究意义**
   - 理论意义：对学科发展的贡献
   - 实践意义：对实际应用的价值

3. **综述目的**
   - 说明为什么需要这篇文献综述
   - 明确要回答的研究问题

4. **结构说明**（可选）
   - 简要告知读者后文组织方式

### 写作示例

```
随着[现实背景]的发展，[研究领域]引起了广泛关注。
尤其是在[具体问题]方面，学者们进行了大量探索（XX, 年份；XX, 年份）。
然而，[指出研究空白或争议]，有必要对现有研究进行系统梳理。
本文将从[理论视角/主题维度]出发，对[研究主题]领域的相关文献进行综述，
以期为后续研究提供理论基础和方向指引。
```

## 二、主体部分（通常占全文70-80%）

### 类型一：按时间顺序组织

适用场景：展示研究演进历程、学科发展脉络

```
├── 2.1 起步阶段（20XX-20XX年）
│   ├── 主要议题
│   └── 研究特点
├── 2.2 发展阶段（20XX-20XX年）
│   ├── 理论突破
│   └── 方法创新
└── 2.3 深化阶段（20XX至今）
    ├── 最新趋势
    └── 研究前沿
```

### 类型二：按主题/议题组织

适用场景：多理论视角并存、需要对比分析

```
├── 2.1 [主题一：理论视角]
│   ├── 2.1.1 观点A及支持文献
│   ├── 2.1.2 观点B及支持文献
│   └── 2.1.3 观点比较与评析
├── 2.2 [主题二：实证研究]
│   ├── 2.2.1 研究方法回顾
│   ├── 2.2.2 主要研究发现
│   └── 2.2.3 研究结论对比
└── 2.3 [主题三：方法论]
```

### 类型三：按研究类型组织

适用场景：不同研究范式差异明显

```
├── 2.1 定量研究
├── 2.2 定性研究
├── 2.3 混合研究
└── 2.4 元分析/系统性综述
```

### 类型四：国内/国外对比

适用场景：存在明显的区域差异

```
├── 2.1 国内研究现状
│   ├── 理论视角
│   ├── 实证发现
│   └── 政策实践
└── 2.2 国外研究现状
    ├── 理论视角
    ├── 实证发现
    └── 政策实践
```

## 三、总结评述部分（通常占全文10-15%）

### 内容要素

1. **综合评述**
   - 现有研究的主要贡献
   - 已达成共识的观点

2. **研究不足**
   - 理论层面：哪些问题尚未解决
   - 方法层面：哪些方法有待改进
   - 实证层面：哪些领域缺乏证据

3. **研究空白**
   - 明确指出现有研究的缺口
   - 阐明这些空白为何重要

4. **本研究定位**
   - 说明本研究将如何填补空白
   - 预告本研究的创新点

### 写作示例

```
综上所述，现有研究在[方面A]和[方面B]取得了重要进展，
但在[研究空白]方面仍存在不足。具体表现为：
第一，[不足1]；第二，[不足2]；第三，[不足3]。
鉴于此，本研究将聚焦于[研究问题]，通过[研究方法]，
旨在[研究目的]，以期为该领域的研究提供[理论/实践]贡献。
```

## 四、其他注意事项

### 段落组织原则

- 每段围绕一个核心观点展开
- 使用主题句（topic sentence）开头
- 文献证据支撑观点，而非淹没观点

### 过渡与衔接

- 段落之间使用过渡句
- 同一主题内文献按逻辑顺序排列
- 时常回顾研究问题，确保不偏离主题

### 图表使用

- 可使用表格汇总核心文献
- 可用图表展示研究方法演进
- 避免为追求形式而添加冗余图表
