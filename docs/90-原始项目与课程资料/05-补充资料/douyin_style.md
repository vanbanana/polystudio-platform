# 抖音内容创作指南

## 抖音内容生态

### 流量池机制

抖音采用逐级流量池推荐：
1. 初始流量：200-500播放
2. 破播放：3000-5000播放（完播率>15%）
3. 中级流量：1W-10W播放
4. 高级流量：50W+播放
5. 热门：500W+播放

### 核心指标权重

| 指标 | 权重 | 说明 |
|------|------|------|
| 完播率 | ★★★★★ | 看完视频的用户比例 |
| 点赞率 | ★★★★☆ | 点赞数/播放量 |
| 评论率 | ★★★★☆ | 评论数/播放量 |
| 转发率 | ★★★☆☆ | 转发数/播放量 |
| 关注率 | ★★★☆☆ | 关注数/播放量 |

## 短视频结构

### 黄金前3秒

决定用户是否继续看下去：

```
✓ 视觉冲击：画面一开始就有亮点
✓ 悬念设置：，激发好奇心
✓ 痛点共鸣：直击用户需求
✓ 反常识：颠覆认知的观点
✓ 问答互动：引发弹幕讨论
```

### 视频脚本结构

```
[开场Hook - 3秒]
制造悬念/痛点/视觉冲击

[核心内容 - 15-45秒]
价值输出，节奏紧凑

[结尾引导 - 3秒]
引导关注/点赞/评论
```

### 时长选择

- **15秒**：信息密集，适合知识类
- **30秒**：平衡型，适合大多数内容
- **60秒**：深度内容，需高完播率支撑

## 标题（描述）规范

### 标题格式

- 字数：建议25字以内
- 重点前置：前10字决定推荐权重
- 关键词：包含1-2个核心关键词
- 引导互动：适当使用问句

### 标题模板

```
[数字+悬念]
3个让你xxx的小技巧，第2个最实用！

[痛点+方案]
总是xxx？试试这个方法！

[反常识]
原来xxx这样做是错的！

[热点+观点]
 xxx上热搜，我的看法是...

[对话型]
xxx说的这句话，你认同吗？
```

### 热门标签

```
#我要上热门 #dou出你的活力 #抖音lovesMe
#今日热门 #挑战全网 #治愈 #正能量
```

## 关键词与话题

### 垂直话题
根据内容选择3-5个相关话题：

```
科技类：#科技 #数码 #手机 #电脑
美食类：#美食 #做饭 #餐饮 #菜谱
美妆类：#美妆 #护肤 #化妆 #变美
```

### 蹭热点技巧
- 发现热点后快速响应（24小时内）
- 与自身内容巧妙结合
- 避免生硬蹭热点

## 内容类型与模板

### 知识干货类

```
脚本结构：
1. 抛出问题/痛点（3秒）
2. 给出解决方案要点（10秒）
3. 实操演示或案例（10秒）
4. 总结+引导互动（5秒）

话术示例：
"很多人都在问xxx问题，今天我来教你..."
"90%的人都做错了，看看你中招了吗？"
"这个xxx技巧，学会能省一半时间"
```

### 好物分享类

```
脚本结构：
1. 场景引入/痛点（3秒）
2. 产品展示+卖点（15秒）
3. 使用效果展示（10秒）
4. 推荐理由+购买链接（5秒）

话术示例：
"这个xxx也太好用了吧！"
"用了半年忍不住安利给你们..."
"学生党/打工人的必备神器！"
```

### 剧情类

```
脚本结构：
1. 场景铺垫+冲突（5秒）
2. 剧情反转/升级（15秒）
3. 结局/彩蛋（3秒）

要点：
- 节奏要快
- 情绪要足
- 超出预期的剧情
```

## 违禁词与限流

### 违禁词

- 绝对化用语：第一、最、顶级、绝对
- 虚假宣传：100%有效、立即见效
- 医疗功效：治病、减肥、增高
- 敏感话题：政治、宗教、争议事件

### 限流原因

- 搬运他人视频
- 频繁删除/隐藏视频
- 诱导互粉互赞
- 账号异常操作
- 内容质量低下

## 发布规范

- 视频比例：9:16（竖版）
- 分辨率：1080x1920 为佳
- 帧率：30fps 或 60fps
- 文件格式：MP4
- 视频大小：建议小于500MB
