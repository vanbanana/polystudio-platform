# 广告文案模板库

## 信息流广告

### 常见版位

| 版位 | 尺寸 | 字符限制 |
|------|------|----------|
| 今日头条 | 1080x1920/1200x628 | 标题20字以内 |
| 腾讯广告 | 1920x1080/1200x628 | 标题20字以内 |
| 百度信息流 | 1920x1080/1200x628 | 标题20字以内 |

### 标题模板

```

[痛点+解决]
还在为xxx发愁？3招帮你轻松搞定

[人群+需求]
xxx人群必看！xxx的绝佳选择

[对比+优势]
xxx与xxx的差距，看完这篇你就懂了

[热点+关联]
最近很火的xxx，真实体验分享

[身份标签]

```

### 描述文案模板

```
第一行：核心卖点（10字以内）
第二行：补充说明/行动引导

示例：
【品质之选】
30年品牌沉淀，专注高端品质
现在咨询，享受专属优惠
```

### 落地页文案

```
[首屏]
主标题：承诺用户能获得什么
副标题：补充说明
CTA按钮：立即领取/立即咨询

[信任背书]
品牌资质/用户评价/数据展示

[核心卖点]
痛点+解决方案

[促销信息]
限时优惠/赠品/活动

[行动号召]
表单/按钮/热线
```

## 抖音/快手广告

### 短视频广告模板

```

[3-15秒] 内容

"原价xxx，现仅需xxx，快去抢购"
"错过等一年，立即点击查看"
```

## A/B 测试建议

### 测试维度

1. **标题**：不同角度（痛点/利益/数字）
2. **图片**：产品图/场景图/对比图
3. **文案长度**：精简版 vs 详细版
4. **CTA**：不同行动号召

### 优化指标

| 指标 | 定义 | 优化方向 |
|------|------|----------|
| CTR | 点击/展示 | 优化素材吸引力 |
| CVR | 转化/点击 | 优化落地页 |
| CPA | 转化成本 | 综合优化 |
| ROI | 投放回报 | 优化投放效率 |
