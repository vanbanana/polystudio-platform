# 小红书种草指南

## 小红书内容生态

### 用户画像

- 核心用户：18-35岁女性为主
- 消费能力：中高端消费意愿强
- 内容偏好：真实、实用、有审美

### 内容定位

| 类型 | 占比 | 特点 |
|------|------|------|
| 干货攻略 | 35% | 有用、能学会 |
| 好物分享 | 30% | 真实体验、客观 |
| 生活方式 | 20% | 有态度、有审美 |
| 情感成长 | 15% | 有共鸣、有价值 |

## 封面设计

### 关键要素

1. **主标题醒目** - 决定点击率
2. **关键词突出** - 便于搜索
3. **视觉风格统一** - 强化记忆
4. **人像/产品突出** - 吸引注意

### 封面模板

```
[单人物]
风格：简约背景+人物主体+大字标题
适用：穿搭、美妆、护肤

[多产品拼图]
风格：整齐排列+使用前后对比
适用：好物推荐、测评

[场景图]
风格：生活场景+产品植入
适用：家居、美食、旅行

[文字为主]
风格：大字标语+简约背景
适用：干货知识、情绪文案
```

### 字体选择

- 简体中文：推荐思源黑体
- 英文/数字：推荐DIN Alternate

## 标题规范

### 标题格式

- 字数：建议18字以内
- 关键词前置：便于搜索
- 适当 emoji：增加活泼感

### 热门标题公式

```
[人群+场景+效果]
学生党平价好物｜均价10元学生党必看！

[数字+卖点]
7个让皮肤变好的小习惯｜学生党也能做到

[对比型]
月薪3千vs月薪3万｜女生的差距在哪里

[疑问型]
毛孔粗大怎么改善？这些方法真的有效！

[情绪型】
绝绝子！这个 xxx 也太好用了吧！

[盘点型】
2024年最值得入手的10件好物
```

## 正文结构

### 种草文模板

```
[开头 - 50字]
场景引入/痛点共鸣

【我的肤质/情况】
（简单说明自身情况，增加可信度）

[主体内容 - 100-300字]
产品/内容详情

✍️ 使用方法：
1. xxx
2. xxx

[总结 - 50字]
推荐理由/使用感受

❤️ 标签
#xxx #xxx #xxx
```

### 干货攻略模板

```
[开头]
为什么要写这篇：解决什么问题

【目录】
1. xxx
2. xxx
3. xxx

[正文分段]
每个部分详细说明

[结尾]
还有什么问题可以问我～

【推荐阅读】
链接到其他相关笔记
```

### 排版技巧

- 善用 emoji 符号：
  - ✨ 种草推荐
  - ✔️ 优点
  - ❌ 缺点
  - ⭐ 强烈推荐
  - 📌 重点
  - 🔗 链接

- 序号清晰：
  - 1️⃣ 2️⃣ 3️⃣
  - ① ② ③

- 段落短小：
  - 每段不超过30字
  - 重要信息单独成段

## 关键词布局

### 标题关键词
- 核心词放前面
- 包含1-2个长尾词

### 正文中关键词
- 首段必含核心词
- 自然出现3-5次

### 标签策略

- 3-5个核心标签
- 1-2个流量型标签
- 1-2个精准型标签
