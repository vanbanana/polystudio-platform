## 1. 小说大纲与世界观设计

### 创作前必问

在开始创作前，先确认以下要素：

1. **题材类型**：玄幻/仙侠/都市/历史/悬疑/轻小说/其他
2. **核心卖点**：主角的特殊能力？爽点？情感？悬疑？
3. **目标字数**：短篇（1-5万）/中篇（10-30万）/长篇（50万+）
4. **主线剧情**：主角要实现什么目标？最终结局是什么？

### 大纲模板

**基础结构（三幕式）：**

```
【开头】建立世界 → 主角出场 → 第一个冲突/金手指
↓
【中段】升级打怪 → 遇到重要配角 → 势力划分 → 中期危机
↓
【高潮】最终对决 → 解决核心矛盾 → 完美/遗憾/开放结局
```

**网文详细大纲模板：**

```markdown
## 《书名》大纲

### 基本信息
- 类型：[玄幻/仙侠/都市/其他]
- 视角：[第一人称/第三人称]
- 预计字数：XX万字
- 核心爽点：[装逼打脸/热血逆袭/甜甜恋爱/烧脑悬疑]

### 世界观设定
【核心规则】
- 这个世界的力量体系：
- 货币/资源体系：
- 势力分布：

【地图/场景】
- 主舞台1：
- 主舞台2：
- 关键地点：

### 人物设定

**主角**
- 姓名：
- 性格：
- 背景：
- 金手指/特殊能力：
- 目标/动机：
- 成长弧光：

**重要配角**（每个配角的戏份和作用要想清楚）
- 配角1：
- 配角2：

**反派/boss**
- 最终反派：
- 中期反派：

### 核心剧情线

**主线**：
第一卷：XXX（XX章）- 引入世界观，主角起步
第二卷：XXX（XX章）- 矛盾升级，组队/势力初现
第三卷：XXX（XX章）- 中期反转，主角陷入危机
第四卷：XXX（XX章）- 最终决战，大结局

### 章节节奏

每2000-3000字需要有一个小高潮/反转/悬念
每10章需要有一个小高潮段落
每30-50章需要有一个大高潮/副本结束
```

### 各类型小说套路参考

**玄幻/仙侠经典套路：**
- 废物主角意外获得金手指（戒指老爷爷/系统/穿越者记忆）
- 退婚流：未婚妻上门退婚，打脸
- 升级流：境界划分为炼气、筑基、金丹、元婴...
- 学院流：进入宗门/学院修炼
- 家族流：家族争斗，逆袭上位
- 洪荒流：盘古开天后裔，夺取先天灵宝

**都市经典套路：**
- 神豪流：系统给钱各种花
- 兵王流：退役特种兵保护总裁/千金
- 修仙归来：都市修仙，扮猪吃虎
- 直播流：直播日常/探险/带货
- 钓鱼/捡漏流：各种奇遇

**历史经典套路：**
- 魂穿/重生到历史人物
- 发明创造推动历史
- 考公/科举之路
- 争霸天下

### 世界观构建技巧

**力量体系设计原则：**
1. 层次分明：境界/等级要有明确划分
2. 有形有色：每个境界有专属特征（如筑基期有异象）
3. 升级可见：主角升级读者要有获得感
4. 限制平衡：防止力量体系崩坏

**地图设计原则：**
1. 功能区分：不同区域有不同功能（修炼区/交易区/险地）
2. 势力制衡：2-4个主要势力相互制衡
3. 逐步展开：从熟悉区域逐步扩展到新地图
4. 有机融合：地图与剧情、人物、力量体系关联

## 2. 角色设定与人物弧光

### 角色构建模板

**主角设定卡：**

```markdown
## 主角：XXX

### 基础信息
- 性别：
- 年龄：
- 身份/职业：
- 性格关键词：

### 外在表现
- 气质/气场：
- 标志性动作：
- 口头禅：
- 标志性装备/特征：

### 内在设定
【过去】
- 童年经历：
- 阴影/创伤：
- 转折点：

【现在】
- 最在意的人：
- 最大的弱点：
- 日常习惯：

【追求】
- 短期目标：
- 终极目标：
- 行为动机：

### 金手指/金大腿
- 金手指类型：
- 获得方式：
- 能力限制：
- 成长空间：

### 人物弧光
- 起点状态：
- 转折事件：
- 终点状态：
```

**配角设定要点：**

每个配角都需要想清楚：
1. **作用**：推动剧情？提供帮助？制造阻碍？丰富世界观？
2. **立场**：队友/对手/中立？会不会变？
3. **记忆点**：读者怎么能记住他？（标志性特点+差异化性格）
4. **结局**：领便当/洗白/继续活跃/隐退？

### 让人物立住的技巧

**给配角加戏的原则：**
- 不要让配角全程围着主角转，他们有自己的生活
- 配角要有自己的目标，与主角目标产生交集或冲突
- 关键时刻靠得住，平时有自己的人物高光

**人物性格一致性：**
- 冷静的人遇到大事不会慌
- 吝啬鬼不会突然大方
- 除非有足够说服力的理由，否则不轻易改变人设

**人物区隔度：**
- 男主身边不能都是暖男/都话唠
- 反派不能都是同一个模板
- 让每个人有独特的记忆点

## 3. 章节正文撰写

### 章节结构模板

**标准网文章节结构（约2000-3000字）：**

```markdown
【开头】（100-300字）
- 承接上文，点明时间地点
- 制造悬念或冲突引入
- 让读者想知道"后来呢"

【铺垫】（500-800字）
- 展现当前局势/困境
- 人物反应和心理
- 为高潮做准备的细节

【高潮/反转】（800-1200字）
- 核心冲突爆发
- 主角应对/反击/选择
- 结果呈现（胜/负/平/留悬念）

【结尾】（100-300字）
- 胜利的收获/失败的代价
- 埋下新悬念
- 引发读者期待下一章
```

### 黄金三章法则

网文前三章决定读者去留，关键要素：

**第一章要点：**
- 主角出场够快（前300字内）
- 困境/金手指/冲突至少出现一个
- 让读者产生期待感

**第二章要点：**
- 矛盾升级或新事件
- 金手指初显威
- 收获/成长

**第三章要点：**
- 第一波小高潮
- 确立主角目标/阵营
- 世界观初步展开

### 写作技巧

**场景切换：**
- 每500-800字可以切换一个小场景
- 大场景切换要留悬念
- 用分割线「——」或「* * *」标记时间/场景跳转

**信息传递：**
- 不要大段堆设定，边用边抛
- 通过角色对话/行动展示信息
- 重要信息可以后面反复提

**节奏把控：**
- 打脸要快准爽，不要拖
- 暧昧要循序渐进，不要急
- 压抑后必反转，保持情绪平衡

**断章技巧：**
- 在高潮处断章
- 在反转处断章
- 在悬念处断章
- 切忌在平铺直叙处断章

## 4. 场景与氛围描写

### 场景描写原则

**为剧情服务的描写才是有效描写：**

不要：
- 大段堆砌华丽辞藻但没有信息量
- 静态描写像风景区介绍
- 无关细节写太多拖慢节奏

要：
- 用场景反映人物心情（情景交融）
- 通过场景预示情节发展
- 场景要有人物活动，不是纯背景

### 各类场景描写参考

**战斗场景：**
```
【低配版】X冲上去，一拳打在Y身上，Y倒飞出去，吐血不止。
【升级版】X体内灵气运转周天，右拳凝聚拳意，如流星坠地般轰向Y胸口。Y仓促间举起双臂格挡，却听"咔嚓"骨碎声响起，整个人如断线风筝般倒飞出去，撞塌三棵古树后方才停下，再也爬不起来。
```

**都市高端场所：**
```
旋转门后是挑高十米的接待大厅，水晶吊灯洒下柔和光芒。大理石地面光可鉴人，空气中飘着淡淡的龙涎香。身穿统一西装的侍者无声地引导方向，角落里摆放着价值不菲的Art Deco风格装饰。xxx踏入此地，恍如隔世——三天前他还在城中村吃泡面。
```

**仙侠修炼时：**
```
灵力如潮水般涌入经脉，xxx引导着它们按照特定周天运转。每运行一个周天，灵力便精纯一分。体内传来清脆的碎裂声，那是瓶颈松动的征兆……又突破了。
```

### 增加画面感的技巧

1. **五感描写**：视觉+听觉+嗅觉+味觉+触觉，三种以上组合更生动
2. **动静结合**：有景色也要有人物活动
3. **对比落差**：主角视角的前后对比
4. **精确数字**：具体数字比笼统的"很多"更有画面感
5. **修辞手法**：比喻、拟人、通感增强感受

## 5. 对白与内心独白

### 对白类型与功能

**推进剧情型：**
```
"三日后的拍卖会，各大家族都会出席。"老者抚摸胡须，"听说压轴的，是一件上古仙宝。"
xxx心中一动："前辈，那仙宝……"
"你不是想知道你是谁吗？"老者似笑非笑，"想知道，就去抢。"
```

**展现人物性格型：**
```
"废物！连这点小事都做不到！"家族子弟趾高气昂。
xxx握紧拳头，指甲嵌入掌心，却一言不发。
三年后，当那人跪在xxx面前求饶时，他终于开口："你当年也是这么说的。"
```
（沉默的人设通过对比反差建立）

**信息透露型：**
通过对话透露世界观的技巧：
- 不直接说设定，而是角色聊天中带出
- 可以用"你连这都不知道？"的桥段
- 重要信息让重要角色来说

### 对白写作技巧

**对话节奏：**
- 不能全是短句，会显得像注水
- 不能全是长句，会显得拖沓
- 长短结合，有问有答

**潜台词：**
```
A："今晚月色不错。"
B："嗯。"
A："一个人赏月挺无聊的。"
B："……所以？"
A："所以，一起？"
```
（实际意思是约人，但不明说）

**避免对白水的问题：**
- 每一句对白都要有信息/情感/人物展示
- 删除后可删除的对白不要留着充字数
- 用行动和描写代替对话能更精炼

### 内心独白使用

**适用场景：**
- 展示角色真实想法（和说出口的话形成对比）
- 渲染情绪氛围
- 解释较为复杂的心理变化

**使用技巧：**
- 用【】或"*内心*"标注
- 不宜过多，每章1-2处即可
- 短小精悍为佳

## 6. 金句与经典台词

### 金句创作技巧

**反转类：**
- 读者以为是A，实际是B
- "我以为你是来杀我的。""我是来救你的。"

**装逼类：**
- "现在道歉，晚了。"
- "你们一起上，我赶时间。"

**情感类：**
- "这世上唯一不会变的，就是一切都会变。"
- "我不怕万人阻挡，只怕自己投降。"

**哲理类：**
- "力量本身不可怕，可怕的是使用力量的人。"
- "有时候，最了解你的反而是敌人。"

### 网文经典爽点金句库

**打脸前奏：**
- "就这？"
- "现在后悔？晚了。"
- "刚才的话，有种再说一遍。"

**绝地反击：**
- "你们以为吃定我了？"
- "这才是我的真正实力。"
- "游戏，现在才开始。"

**收获时刻：**
- "这波赚大了。"
- "属于我的，谁也抢不走。"
- "从今天起，这片天，我来撑。"

**情感时刻：**
- "只要你需要，我就在。"
- "这次，换我来守护你。"
- "什么都不重要，重要的是你没事。"

### 金句使用原则

1. **放在关键位置**：高潮点、断章处、转折点
2. **配合情境**：情绪到了才有力
3. **控制频率**：每章最多2-3句金句，多了就不值钱了
4. **符合人设**：不同角色说不同风格的话

## 7. 卡文与情节推进

### 卡文常见原因与对策

**原因1：不知道下一步写什么**
- 对策：翻回大纲，看看下一阶段目标是啥
- 技巧：制造一个新冲突/意外事件

**原因2：剧情走向不确定**
- 对策：列出3个可能选项，问自己哪个最爽
- 技巧：参考同类作品的常见选择

**原因3：卡在高潮如何设计**
- 对策：先跳过写后面的事，再回来填
- 技巧：阅读类似高潮章节找灵感

**原因4：配角不知道干啥**
- 对策：给配角安排独立支线
- 技巧：让配角也有自己的目标

### 快速推进剧情的技巧

**事件驱动法：**
```
当前状态：A和B在讨论修炼
↓加入意外
突然传来消息：C家族大打出手，双方死伤惨重
↓加入选择
主角面临选择：去还是不去？
↓加入后果
去了可能得罪人，不去可能错过机遇
```

**层层递进法：**
```
第一层：小BOSS出现，打脸
↓
第二层：发现背后有人，调查
↓
第三层：引出中BOSS，冲突升级
↓
第四层：发现惊天阴谋
```

**回忆杀/解释杀（慎用）：**
- 主角遇到强者，回想起过去的屈辱
- 遇到神秘事物，引入新设定
- 触发某个事件，引出隐藏人物

### 断更好恢复

如果断了很久想恢复连载：
1. 写章前情提要
2. 简单回顾前情
3. 快速进入新冲突
4. 让配角提醒读者主角的目标

## 8. 文笔润色与风格优化

### 各类型文风特点

**小白文（轻松易读）：**
- 短句为主
- 较少修饰
- 节奏明快
- 爽点直接

**老白文（有一定文笔）：**
- 描写丰富
- 伏笔较多
- 思考深入
- 节奏稍慢

**文青/二次元：**
- 内心戏多
- 吐槽向
- 玩梗
- 情绪细腻

### 润色技巧

**删除冗余：**
- 副词过多（非常、特别、极其）
- 重复表达（他走了出去，离开了这里）
- 语气词滥用（啊、呀、呢）

**增加质感：**
- 用具体替代笼统
- 用动词替代形容词
- 用短句制造节奏感

**示例：**
```
原文：他非常生气地冲了过去 使出很强的招数把敌人打跑了
润色：他冲上去 一拳轰出 灵力如火山喷发 对方甚至来不及反应便被砸飞出去 挣扎两下 没了气息
```

### 网文写作禁忌

1. **水文**：大段无关描写/对话
2. **OOC**：人物性格前后不一致
3. **崩战力**：力量体系前后矛盾
4. **毒点**：圣母/绿帽/强行虐主（大男主爽文）
5. **断更**：突然消失不填坑
6. **虎头蛇尾**：大纲和结尾差距太大

### 提升建议

1. **扫榜学习**：看同类型头部作品，学习它们的节奏
2. **记录灵感**：随时记录想到的片段/金句/点子
3. **保持更新**：日更4000+是基本功
4. **重视读者**：看评论，理解读者想看什么
5. **写完为王**：不要太监，哪怕写得不满意也要完本
